# ToDoApp — Список завдань

Мобільний застосунок «Список завдань» (To-Do), створений на **React Native** з використанням **Expo**.

Залікова робота з дисципліни «Розробка інтерфейсів користувача».

## Можливості

- ➕ Додавання нових завдань
- 🗑️ Видалення завдань
- ✅ Позначення завдань як виконаних / невиконаних
- 🔎 Фільтрація завдань: **Всі**, **Активні**, **Виконані**
- 💾 Збереження даних у локальному сховищі **AsyncStorage** (завдання зберігаються після закриття додатку)

## Технології

- React Native
- Expo (SDK 54)
- @react-native-async-storage/async-storage

## Вимоги

Перед запуском має бути встановлено:

- [Node.js](https://nodejs.org) (версія LTS)
- Додаток **Expo Go** на телефоні (з App Store або Google Play)

## Запуск

1. Клонувати репозиторій:

   ```bash
   git clone https://github.com/MaksYaloza/ToDoApp.git
   cd ToDoApp
   ```

2. Встановити залежності:

   ```bash
   npm install
   ```

3. Запустити проєкт:

   ```bash
   npx expo start
   ```

   Якщо телефон і комп'ютер не з'єднуються по локальній мережі, запустіть у режимі тунелю:

   ```bash
   npx expo start --tunnel
   ```

4. Відсканувати QR-код із терміналу:
   - **iPhone** — звичайною камерою, потім відкрити в Expo Go;
  

> Комп'ютер і телефон мають бути підключені до однієї Wi-Fi мережі (для звичайного режиму запуску).
